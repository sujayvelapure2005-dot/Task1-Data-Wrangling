\# Data Dictionary



| Column Name | Description |

|-------------|-------------|

| ORDERNUMBER | Unique order identifier |

| QUANTITYORDERED | Number of products ordered |

| PRICEEACH | Price of each item |

| ORDERLINENUMBER | Order line number |

| SALES | Total sales value |

| ORDERDATE | Date when order was placed |

| STATUS | Order status |

| QTR\_ID | Quarter of the year |

| MONTH\_ID | Month of order |

| YEAR\_ID | Year of order |

| PRODUCTLINE | Product category |

| MSRP | Manufacturer suggested retail price |

| PRODUCTCODE | Unique product code |

| CUSTOMERNAME | Name of the customer |

| PHONE | Customer phone number |

| ADDRESSLINE1 | Customer address |

| CITY | Customer city |

| STATE | Customer state |

| POSTALCODE | Zip code |

| COUNTRY | Country of customer |

| TERRITORY | Sales territory |

| CONTACTLASTNAME | Customer last name |

| CONTACTFIRSTNAME | Customer first name |

| DEALSIZE | Size of the deal |
